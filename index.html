Acerca del Proyecto

¿Qué es Jira Calendar Converter?

Esta es una herramienta puente que facilita la visualización y sincronización de tareas entre Jira y Google Calendar. Aunque Jira permite exportar sus issues en un archivo CSV, este formato crudo carece de compatibilidad nativa fluida con plataformas de calendario corporativas. Esta aplicación procesa, estandariza y traduce esa data a un formato perfectamente digerible de importación (Google).

¿Por qué se creó?

Manejar sprints, entregables y seguimientos requiere una visión cronogramada constante. Copiar manualmente cada tarea a un calendario personal o de equipo para agendar bloques de tiempo es sumamente tedioso y propenso a omisiones humanas. Esta aplicación nació como una solución para automatizar y masificar ese puente visual sin fricciones de formato.

Privacidad y Seguridad

Su código está fundamentado bajo el principio de procesamiento local. Se ejecuta enteramente en el navegador del usuario y su ambiente anfitrión local. Los archivos o exportes CSV de la empresa nunca se envían ni se retienen en ningún servidor externo, quedando todas la información e historiales confidenciales protegidos dentro de tu propio equipo de trabajo.

El servidor local no está corriendo

Para conectar con la API de Jira y evitar bloqueos de seguridad del navegador, necesitas levantar el servidor ligero de Node.js incluido en la carpeta.

Pasos para iniciarlo:

  1. Abre una terminal (CMD o PowerShell) en la carpeta de este proyecto.
  2. Opcional (solo la primera vez): Instala las dependencias ejecutando:
    npm install
  3. Luego arranca el servidor ejecutando:
    node server.js

Una vez lo inicies, haz clic en Verificar o recarga esta página en http://localhost:3000.

Guía de uso
1

Exportar desde Jira

  1. Ve a tu proyecto en Jira y abre la vista de Lista o Backlog.
  2. Aplica los filtros que necesites (sprint, épica, asignado).
  3. Haz clic en Exportar → Exportar a CSV.
  4. Asegúrate de seleccionar "Todos los campos".
2

Opciones y Carga Múltiple

  • La herramienta detecta automáticamente el idioma de las columnas.
  • Puedes arrastrar más de un archivo CSV para fusionarlos.
  • Revisa la gráfica de distribución antes de exportar.
3

Crear un Calendario Nuevo

  1. Abre Google Calendar.
  2. Junto a Otros calendarios, haz clic en + → Crear un calendario.
  3. Ponle nombre (ej. "Tareas Jira") y haz clic en Crear.
  4. (Así puedes borrarlas todas fácilmente si es necesario).
4

Importar el CSV generado

  1. Haz clic en → Configuración.
  2. Selecciona Importar y exportar en el menú lateral.
  3. Sube el CSV que descargaste de esta herramienta.
  4. Importante: selecciona el calendario nuevo que creaste.
  5. Haz clic en Importar.
